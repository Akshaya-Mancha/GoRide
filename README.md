# 🚗 GoRide - EV Rental System

## 📌 Description

GoRide is an Electric Vehicle (EV) rental system that allows users to rent eco-friendly vehicles for their travel needs. The system helps users book, manage, and track EV rentals efficiently.

## 🚀 Features

* User registration and login
* Browse available EV vehicles
* Rent/Book EVs
* View booking details
* Manage rental history
